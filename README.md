# St. Thomas Malankara Syrian Catholic Church Website

A modern, responsive website for St. Thomas Malankara Syrian Catholic Church, Nalanchira, Kerala, India. This website showcases the church's rich heritage, community activities, and provides essential information for parishioners and visitors.

## 🌟 Features

### Modern Design
- **Responsive Layout**: Optimized for all devices (desktop, tablet, mobile)
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Beautiful Typography**: Google Fonts (Playfair Display & Poppins)
- **Color Scheme**: Professional church-themed colors with gold accents

### Key Pages
1. **Home Page** (`index.html`)
   - Hero carousel with church images
   - About section with church history
   - Services overview
   - Upcoming events
   - Call-to-action sections

2. **About Us** (`about.html`)
   - Church history timeline
   - Mission & Vision statements
   - Core values
   - Interactive timeline design

3. **Church Personnel** (`personnel.html`)
   - Clergy information
   - Parish council members
   - Ministry leaders
   - Professional staff cards

4. **Contact Us** (`contact.html`)
   - Contact form with validation
   - Service times
   - Location information
   - Interactive map
   - Contact details

5. **Gallery** (`gallery.html`)
   - Photo gallery with lightbox
   - Category filtering
   - Video gallery section
   - Responsive grid layout

6. **Organizations** (`organizations.html`)
   - Church organizations
   - Ministry groups
   - Community activities

7. **Prayer Groups** (`prayer_groups.html`)
   - Prayer group information
   - Meeting schedules
   - Group descriptions

## 🛠️ Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS Grid, Flexbox, and custom properties
- **Bootstrap 5**: Responsive framework
- **JavaScript**: Interactive functionality
- **Font Awesome**: Icons
- **Google Fonts**: Typography
- **AOS (Animate On Scroll)**: Scroll animations
- **GLightbox**: Image lightbox gallery

## 🚀 Getting Started

1. **Download/Clone** the website files
2. **Open** `index.html` in a web browser
3. **Navigate** through the pages using the navigation menu
4. **Customize** content, images, and colors as needed

## 🔧 Customization

### Updating Content
- Edit the HTML files to update text content
- Replace images in the `images/` folder
- Update contact information in the footer

### Changing Colors
- Modify CSS custom properties in the `:root` selector
- Update color values to match your brand

### Adding Pages
- Create new HTML files following the existing structure
- Include the same header, navigation, and footer
- Add new menu items to the navigation
